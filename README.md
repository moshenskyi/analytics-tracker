
# analytics-tracker

## Install

1. In your root `build.gradle` add `jitpack` repository:

```
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```

2. In your `/app/build.gradle` add dependency:

```
dependencies {
	implementation 'com.github.moshenskyi:analytics-tracker:$latestVersion'
}
```
